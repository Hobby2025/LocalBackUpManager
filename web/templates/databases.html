    데이터베이스 추가
    데이터베이스 종류를 선택하면 기본 포트가 자동 설정됩니다.
    SQLite의 경우 파일 경로를 입력하세요.
    기본 포트가 자동 설정됩니다.
    연결할 데이터베이스 이름을 입력하세요.
    데이터베이스 접속 사용자명을 입력하세요.
    수정 시 비밀번호를 비워두면 기존 비밀번호를 유지합니다.
    SSL/TLS 연결 보안 수준을 선택하세요.
    📅 백업 스케줄
    전체 데이터베이스 백업 주기를 설정합니다.
    변경된 데이터만 백업하는 주기를 설정합니다.
    백업이 시작될 시간을 설정합니다.
    체크 해제 시 수동 백업만 가능합니다.
    🗜️ 압축 및 암호화
    백업 파일 압축 방식을 선택합니다.
    압축 품질과 속도의 균형을 조절합니다.
    백업 파일을 암호화하여 보안을 강화합니다.
    ⚠️ 이 비밀번호를 분실하면 백업을 복원할 수 없습니다.
    🗄️ 보존 정책
    일간 백업을 보관할 일수입니다.
    주간 백업을 보관할 주수입니다.
    월간 백업을 보관할 개월수입니다.
    단일 백업 파일의 최대 크기를 제한합니다.
    보존 정책에 따라 오래된 백업을 자동으로 삭제합니다.
