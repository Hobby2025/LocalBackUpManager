🗄️ 데이터베이스 타입을 선택하세요

백업하려는 데이터베이스의 종류를 선택해주세요. 각 DB별로 최적화된 설정을 제공합니다.

추천
🐘
PostgreSQL

고성능 오픈소스 관계형 데이터베이스

  • 뛰어난 확장성
  • 고급 SQL 기능
  • 우수한 압축률
  • 병렬 백업 지원
🐬
MySQL

세계에서 가장 인기 있는 오픈소스 DB

  • 빠른 성능
  • 쉬운 관리
  • 광범위한 지원
  • 안정적인 백업
📄
SQLite

경량 파일 기반 데이터베이스

  • 설정 불필요
  • 초고속 백업
  • 단일 파일
  • 개발/테스트 최적
💡 선택 가이드:
  • PostgreSQL: 대용량 데이터, 복잡한 쿼리, 높은 동시성이 필요한 경우
  • MySQL: 웹 애플리케이션, 빠른 읽기 성능이 중요한 경우
  • SQLite: 소규모 애플리케이션, 개발/테스트 환경, 임베디드 시스템
